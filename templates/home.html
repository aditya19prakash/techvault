These endpoints are the starting point for all new users.

POST /user/register/ ‚Äì Create a new user account
POST /user/login/ ‚Äì Login and receive JWT tokens
POST /user/logout/ ‚Äì Logout (token-based)

üìö Resource Endpoints

GET /resources/ ‚Äì Fetch all shared resources
POST /resources/ ‚Äì Create a resource (JWT required)
GET /resources/<id>/ ‚Äì Retrieve resource details

üí¨ Comments & Voting

POST /resources/<id>/comment/ ‚Äì Add a comment
POST /resources/<id>/vote/ ‚Äì Upvote or downvote

üöÄ How a Beginner Should Use This API

  1. Register using POST /user/register/
  2. Login using POST /user/login/ to receive a JWT token
  3. Attach the token in headers: Authorization: Bearer <access_token>
  4. Access protected endpoints such as creating resources, commenting, or voting
